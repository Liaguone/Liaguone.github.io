油管up推荐

2023-09-12
阅读时长: 6 分钟

建议大家学不进去的时候都看看油管博主

每次学习没有动力的时候，我都喜欢看一些学习博主的vlog,现在真的很喜欢看油管学霸的vlog，他们都是世界名校的学生，看他们学习真的很治愈，同时他们会分享很多学习的方法，有时候真的会惊叹：还能这样学？

计算机

Abdul Bari

▪️这个老哥是真的想教会咱的计算机🥺！！各个章节想看什么可以直接搜索，涵盖很多内容

John Fish

哈佛小哥，计算机科学学霸 分享学习和效率方法，此外也会分享一些书单和阅读心得 ◾️《如何利用心流从不爱学习到享受学习》 ◾️《成为高效学习者的4个习惯》

Telusko

▪️内容主要涉及python，java，区块链等方面，一般一个视频时长10-20分钟左右。💻很适合零基础想学习python的同学

Jenny’s Lectures CS IT

▪️对CS专业的同学很友好，赶紧来抱大腿！不过视频里经常英语夹杂着印度

Tanmay Bakshi

印度天才少年，9岁发布自己的第1️⃣个苹果应用程序，12岁成为IBM的荣誉顾问，用通俗易懂的语言讲解各种编程知识

my code school

现在内外都能看到他的课程，虽然已经不再更新，但是现有的课程内容已经特别丰富。

programming with mosh

有javaScript、C语言等内容，非常基础，很适合新入门的小白学习

Free Code Camp

Free Code Camp 是一个非盈利组织网站，其目的是帮助人们学习代码，完全免费。

The New Boston

The New Boston 是 YouTube 上最受欢迎的网络开发频道之一

Computerphile

涵盖非常广泛的技术概念，从计算机硬件、软件到密码学、数据科学、人工智能等各种主题

Traversy Media

Traversy Media 上的视频对作为 Web 开发人员具有一定专业知识的个人更有帮助

Clever Programmer

由Rafeh Qazi创建的Clever Programmer，为用户提供了最好的Web开发课程

CS Dojo

CS Dojo 是在谷歌任职的程序员 YK Sugi 所创建，他的视频集包含有关数据科学、ML、Django、Web 开发框架和创造性的问题解决

ProgrammingKnowledge

一个关于前端和后端的技术频道。它的一些可成非常适合那些刚刚开始学习编程的人

Coding Train

Coding Train 是由程序员 Daniel Schiffman 主持的一个受欢迎且有趣的频道

Derek Banas

它还提供前端和后端的全方位Web开发技术教程

数学

Numberphile

数学大佬都在看的学习频道，探索各种数学知识。在这里，只有原汁原味的硬核讲解。

Khan Academy

▪️这个大家应该都知道！里面有很多的印度老师，学高数、概率这些都可以来看！

英语

English with Lucy 跟着Lucy学英语

颜值高，专注英式英语、高级词汇，在油管人气非常旺。up: 跟着Lucy学英语

Learn English with Bob the Canadian

和加拿大鲍勃学英语

这个频道非常火爆，是我公司做亚马逊的小哥推 荐的，语速慢，简单实用。up: 听学英语吧

Sydney Serena

一位教英语的小姐姐，英语口语说的真的很纯正 ◾️《跟着油管vlog学英语》

BBC 官 方英标教程

想提升发音的同学必看，BBC官 方出品，这个系列很经典，up：今天你又没学习

speak English with Vanessa

6.25m

Learn English with TV series

这个系列的《生活大爆炸》，《艾伦秀》，《老友记》以及电影系列都收获了很多订阅者的喜欢。

哥伦布 Little Columbus

这是一位来自台湾的小哥，主要分享英文发音，日常英语的用法，有着很魔性的开场白，很有趣味性，视频内容为中英混和，对小白也很友好！

Maxxter English

这是一位生活在美国的小姐姐，中英文都特别好，她的视频涉及生活话题，还有一些贴合实事的话题，实用性超高！她的中英字幕很准确，而且还会用正常速度和慢速朗读，确保你能听清发音！ ·

Mad English

这个博主是加拿大人，会教授大家一些基础的生活会活英国，他的视频是PPT➕真人讲解，视频纯英文，但内容不难，可以锻炼自己的听力！ ·

JamesEngvid

看他的视频就像在看脱口秀，博主是一个加拿大黑人，很会卖萌，上课会板书，一边讲解一边补充，语速偏快，更偏向教一些native speaker的用法，还有当堂练习，检验学习成果！ ·

7ESL Learning English

唯一一个没有人设的英语学习博主，ESL的全科是English as Second Language，对外国人很友好，内容丰富，有语法、单词、日常会话……

医生

Ninja Nerd

▪️真的是医学生的救命prof！👨‍⚕️创始人Zach Murphy和他好朋友 Robert和 Christian，共同创办了频道Ninja Nerd，希望借此以一种影响更大的方式普惠医学生

Ali Abadaal

剑桥医学院毕业的学霸博主，在油管上有超过500万粉丝 做博主、做生意、学习、写书都能兼顾，分享的学习方法、效率、个人成长等干货都很受用！ ◾️《用一年的时间彻底改变人生，活成自己想要的样子》 ◾️《用5种方法找到人生方向》 ◾️《3.5小时沉浸式学习，番茄时钟+写作业音乐，集中注意力》

Elizabeth Flips

伦敦国王学院的医学生，同时是油管百万博主，画家，作家，博客主理人 她真的对于传统学习/效率方法有着独特的视角，看她的视频能让你接纳自己，找到自己的节奏与方法 ◾️《没有天生的笨人，如何用简单的方式做复杂的事》 ◾️《学霸的学习方法总结！告诉你最强大的学习动力！》

Elly

又是一位医学生，她的视频真的很给大家续学习动力 看她的视频就是喝咖啡☕️，学习 ◾️《韩国女医学生紧张的10天期末考试vlog》 ◾️《心脏病考试前72小时》
