  • 计算机系统中，数值用补码来表示和存储
  • 使用补码，可以将符号位和数值域统一处理:同时，加法和减法也可以统一处理。
  • 原码:最高位表示符号位，0表示正数，1表示负数，其余位表示为整数的二进制数。
  • 反码:正数的反码与原码相同，负数的反码反码是在原码的基础上对除符号位之外的位取反，0变1，1变0。
  • 补码:正数的反码与原码相同，负数的补码是在反码的基础上，对最低位+1。

python序列

序列:一块可存放多个值的连续内存空间，这些值按一定顺序排列，可通过每个值所在位置的编号(称为索引)访问它们。

序列中的每个值都有对应的位置值，称为索引，第一个索引是0，第二个索引是1，依此类推。

序列是Python中最基本的数据结构。 常用序列:列表、元组、字典、集合、字符串

序列通用操作:索引、长度、组合(列相加)、重复(乘法)、分片、检查成员、遍历、最小值和最大值。

# 定义一个包含字符串的列表 a
a = ['tree', 'tree', 'till', 'you']
# 在索引位置7插入一个 '.'，由于索引超出范围，它会在列表的末尾插入该元素
a.insert(7, '.')
print(a)  # 输出：['tree', 'tree', 'till', 'you', '.']
# 扩展列表 a，添加两个新的元素 'put' 和 'down'
a.extend(['put', 'down'])
print(a)  # 输出：['tree', 'tree', 'till', 'you', '.', 'put', 'down']
# 移除列表的最后一个元素（'down'）
a.pop()
print(a)  # 输出：['tree', 'tree', 'till', 'you', '.', 'put']
# 移除列表中第一次出现的 'tree'
a.remove('tree')
print(a)  # 输出：['tree', 'till', 'you', '.', 'put']
# 清空列表 a 中的所有元素
a.clear()
print(a)  # 输出：[]
# 重新定义列表 a
a = ['be', 'give', 'you', 'a', 'be', 'that']
# 计算 'be' 在列表 a 中出现的次数
b = a.count('be')
print(b)  # 输出：2
# 查找 'be' 在列表 a 中第一次出现的索引
c = a.index('be')
print(c)  # 输出：0
# 排序列表 a
a.sort()
print(a)  # 输出：['a', 'be', 'be', 'give', 'that', 'you']
# 反转列表 a
a.reverse()
print(a)  # 输出：['you', 'that', 'give', 'be', 'be', 'a']
# 创建列表 a 的副本 d
d = a.copy()
print(d)  # 输出：['you', 'that', 'give', 'be', 'be', 'a']

切片操作

  • 切片是Python序列的重要操作之一
  • 适用于列表、元组、字符串、range对象等类型
  • 用于截取序列中的任何部分，得到一个新序列
  • 也可以用于修改和删除列表中部分元素，或增加元素

切片:2个冒号分隔的3个数字

  • 第一个数字表示切片开始位置(默认为0)
  • 第二个数字表示切片截止(但不包含)位置(默认为列表长度)
  • 第三个数字表示切片的步长(默认为1)，当步长省略时可以省略后一个冒号

  • 定义函数时的参数称为形式参数(简称形参);调用函数时的参数称为实在参数(简称实参)
  • 一般类型的数据作为参数相当于传值调用，可变类型的序列作为参数相当于传地址调用
  • 定义函数时不需要指定参数的类型和返回值的类型
  • 普通参数，默认值参数，关键参数，可变长度参数

return语句

  • return语句用于函数体，可结束函数的运行并返回return后的表达式
  • 没有return语句的函数，或者return语句后没有表达式的函数，其返回值为None
  • 任何函数都可以作为表达式的一份子，也可以作为独立语句调用

变量作用域

  • 变量起作用的代码范围称为变量的作用域，不同作用域内同名变量之间互不影响
  • 函数内部定义的变量一般为局部变量，而不属于任何函数的变量一般为全局变量
  • 局部变量的引用速度比全局变量快，应优先考虑使用
  • 尽量避免使用全局变量
  • 一个变量已经在函数外定义，如果在函数内需要使用这个变量，并将改变变量的结果反映到函数之外，可以在函数内用global声明这个变量
  • 在函数内部直接使用global关键字可以将一个变量声明为全局变量

  • map() 函数是 Python 中的一个内建函数，它用于将指定函数应用于给定可迭代对象（如列表、元组等）的每个元素，并返回一个迭代器（在 Python 3 中是 map 对象）。你可以使用 list() 或 for 循环等方式将其转换为列表或其他类型。

  • map(function, iterable, ...)

    function：一个函数，它将应用于 iterable 中的每个元素。这个函数可以是标准的函数、lambda 表达式或其他可以接受单个参数的函数。

    iterable：一个或多个可迭代对象（如列表、元组、集合等）。如果有多个 iterable，function 必须能够接受多个参数（即函数的参数数量应该与可迭代对象的数量匹配）。

    map() 返回一个 迭代器，你需要将其转换为一个数据类型（如列表、元组等）来查看结果。
